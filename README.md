# QR-Generator
QR Code Generator 🧲


## About this project 🚀
In this QR Code Generator app, users can enter a text or URL to generate a QR code for it. (It is a QR code generator app, not a QR code scanner)


## Technologies 🛠️
* `HTML`
* `CSS`
* `JavaScript`
